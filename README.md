# Saunter

## Getting Started

To get started with the project, follow these steps:

1. **Clone the repository:**
    git clone https://github.com/MinenkoVladlen/saunter.git
    cd saunter

2. **Install dependencies:**
    npm install

3. **Run the project:**
    npm run dev

## Running Tests

To run the tests for the project, follow these steps:

1. **Run the tests:**
    npm run test
